# Hamro Samajik Hub

A calm, minimal social media web app built with plain HTML, CSS, and JS.

| File | Description |
|------|-------------|
| `pages/index.html` | Home feed with posts and stories |
| `pages/trending.html` | Photo explore grid (Instagram-style) |
| `pages/profile.html` | User profile page |
| `pages/login.html` | Login page |
